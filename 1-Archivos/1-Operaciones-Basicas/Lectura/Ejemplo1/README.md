# Lectura de un archivo de personas
Utiliza como base el 'Ejemplo 2' de la sección 'Creación'
##### Este ejemplo realiza las siguientes acciones:

1. Utiliza un procedimiento 'recorrer_archivo_personas' para:
    1. Obtener los datos de un registro persona
    2. Imprimirlos en la consola

#### Demuestra: 
* Que es necesario pasar como 'var' a un parámetro de tipo file
* El archivo debe existir para poder ser leído (si no, reset() dará error)
* No se realiza el assign. Eso se da por hecho.
* El tipo de dato que usamos para leer debe ser el mismo que el que contiene el archivo
* La lectura del archivo se realiza de registro a registro
* La impresión en consola es campo a campo
