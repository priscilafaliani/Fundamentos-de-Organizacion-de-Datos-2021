# Creación de un archivo de registros de personas

###### Este ejemplo realiza las siguientes acciones:

1. Definir un tipo file, que guardará registros como sus elementos.

2. Leer un nombre para el archivo que se va a crear.

3. Crear el archivo.

4. Leer datos de personas en registros y escribir los registros en el archivo creado.

5. La lectura/escritura de personas finaliza cuando se ingresa el dni ''.

6. Cierra el archivo y finaliza.
