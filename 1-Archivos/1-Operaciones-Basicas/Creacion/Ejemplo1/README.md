# Creación de un archivo de enteros

###### Este ejemplo realiza las siguientes acciones:

1. Definir un tipo file, que guardará integer como sus elementos.

2. Leer un nombre para el archivo que se va a crear.

3. Crear el archivo.

4. Leer números enteros y escribirlos en el archivo creado.

5. La lectura/escritura de números finaliza cuando se ingresa el número 0.

6. Cierra el archivo y finaliza.
