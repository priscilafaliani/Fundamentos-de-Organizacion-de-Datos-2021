# Modificación de un archivo de personas
Utiliza como base el 'Ejemplo 2' de la sección 'Creación'
##### Este ejemplo realiza las siguientes acciones:

1. Utiliza el procedimiento 'actualizar_salario' para actualizar el salario de todos las personas en un porcentaje dado
    1. Abre el archivo que recibe
    2. Mientras no llegue al EOF
        1. Obtiene un registro persona
        2. Actualiza el salario
        3. Escribe el registro nuevamente en el archivo
    3. Cierra el archivo.
