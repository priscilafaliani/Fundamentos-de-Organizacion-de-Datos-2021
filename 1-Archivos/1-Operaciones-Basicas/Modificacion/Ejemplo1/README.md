# Modificación de un archivo de personas
Utiliza como base el 'Ejemplo 2' de la sección 'Creación'
##### Este ejemplo realiza las siguientes acciones:

1. Utiliza el procedimiento 'agregar_personas' para agregar elementos al final de un archivo
    1. Abre el archivo que recibe
    2. Lee los datos de una persona
    3. Mientras el dni sea distinto de ''
        1. Escribe el elemento al final de archivo.
        2. Lee los datos de una persona.
    4. Cierra el archivo
