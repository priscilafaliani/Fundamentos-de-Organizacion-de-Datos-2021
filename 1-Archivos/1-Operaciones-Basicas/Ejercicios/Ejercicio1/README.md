# Ejercicio 1
Ejercicio 2, respecto al libro "Introducción a las Bases de Datos - Rodolfo Bertone"

## Descripción
Realice un algoritmo que permita guardar en un archivo todos los productos que actualmente se encuentran en venta en un determinado negocio. La información que es importante considerar es: nombre del producto, cantidad actual, precio unitario de venta, tipo de producto (comestible, de limpieza, vestimenta).

_Son demostraciones muy básicas, falta el manejo de excepciones y demás pero en fin_
